# Security Lab Simulator

Aplikasi web frontend-only berbasis React + Vite untuk edukasi dan simulasi keamanan siber secara aman.

## Stack
- React + Vite
- Tailwind CSS
- React Router
- Zustand (state + localStorage persistence)
- Recharts
- Framer Motion

## Jalankan lokal
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy ke Vercel
1. Push repository ke GitHub.
2. Import project di Vercel.
3. Framework preset: **Vite**.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Deploy.

"Platform ini hanya untuk edukasi, simulasi, dan pembelajaran keamanan siber di lingkungan yang sah. Tidak dirancang untuk menyerang, menguji, atau berinteraksi dengan sistem nyata tanpa izin."
